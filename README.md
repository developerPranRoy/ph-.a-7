# KeenKeeper — Keep Your Friendships Alive

A beautiful and intuitive web application to help you maintain meaningful relationships by tracking your last interactions with friends and family.

![KeenKeeper Dashboard](https://via.placeholder.com/800x400?text=KeenKeeper+Dashboard)

## ✨ Features

- **Responsive Design** – Works perfectly on Mobile, Tablet & Desktop
- **Friend Management** – Add and track friends with personal details
- **Smart Status System** – Visual indicators (On-Track, Almost Due, Overdue)
- **Quick Check-In** – Log calls, texts, and video calls with one click
- **Interactive Timeline** – Complete history of all interactions
- **Friendship Analytics** – Beautiful pie chart showing interaction types
- **Toast Notifications** – Real-time feedback
- **Loading States & 404 Page** – Polished user experience

## 🛠️ Technologies Used

- **Next.js 16** (App Router)
- **React**
- **Tailwind CSS**
- **Recharts** (for Analytics Pie Chart)
- **Lucide React** (Icons)
- **Sonner** (Toast Notifications)
- **LocalStorage** (Data Persistence)
